# Cursor による IaC 実装

このディレクトリには、Cursor を使用して開発されたAWS IaCの実装が含まれます。

## Cursor について

Cursor は AI を統合したコードエディタで、VS Code をベースにAIペアプログラミング機能を提供します。リアルタイムでのコード提案、編集、リファクタリングをサポートします。

## 特徴

- **IDE統合**: フル機能のエディタ環境でAI支援
- **リアルタイム提案**: タイピング中のインテリジェントな補完
- **コンテキスト認識**: プロジェクト全体を理解した提案
- **マルチファイル編集**: シームレスなファイル間の編集
- **チャット機能**: エディタ内での対話的な開発

## セットアップ

1. Cursor のインストール
   - [公式サイト](https://cursor.sh/)からダウンロード

2. プロジェクトを開く
```bash
cursor .
```

3. AI機能の有効化
   - Settings > AI > モデルを選択

## 開発ワークフロー

### 1. 仕様ファイルの確認
- `.kiro/specs/` ディレクトリの仕様を Cursor で開く
- AI チャットで要件を確認

### 2. IaC実装
- Cmd/Ctrl + K でAI編集モード起動
- 自然言語で実装指示
- リアルタイムでコード生成・編集

### 3. レビューと改善
- AI チャットでコードレビュー依頼
- 提案された改善を適用

- `Cmd/Ctrl + L`: AIチャットパネル
